AI Resume Critiquer

To use this application, get an OpenAI API key from https://platform.openai.com/api-keys and create a .env file in the project folder with OPENAI_API_KEY=your_key_here.

Steps:
1.  Go to https://docs.astral.sh/uv/ and install uv.
2.  Create a folder and type the command "uv init ." into the terminal.
3.  Type "uv add streamlit openai PyPDF2 python-dotenv" to add imports.
4.  Finally, run "uv run streamlit run main.py" to use the AI image detector.
